Getting Started¶

Requirements¶

  • Python 3.8+

Installation¶

The armasec package can be installed like any other python package. It is available on PyPI

Poetry¶

To install via poetry, simply run:

$ poetry add armasec

Pip¶

To install directly with pip, simply run:

$ pip install armasec
